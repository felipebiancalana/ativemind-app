# Ativemind App

Este é o projeto simplificado para rodar na Lasy, com preview em tempo real.

## Estrutura do projeto

- `src/app/page.jsx` → página principal
- `src/app/layout.jsx` → layout básico
- `style.css` → estilos globais
- `package.json` → dependências mínimas

## Como usar

1. Abra o projeto na Lasy.
2. As alterações em `page.jsx` ou `style.css` aparecerão **em tempo real** no preview.
3. Para salvar alterações permanentemente, clique em **Commit Changes**.
